# 建议常用命令（npm）

## 开发
- npm run dev
- npm run build
- npm run start
- npm run preview

## 质量门槛
- npm run check  # next lint + tsc --noEmit
- npm run lint
- npm run lint:fix
- npm run format:check
- npm run format:write

## 数据库（Drizzle）
- ./start-database.sh
- npm run db:push
- npm run db:migrate
- npm run db:generate
- npm run db:studio
